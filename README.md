## ✨ What is this?

The official personal brand website for Emil Heyerdahl — a professional racing driver. Cinematic, high-impact design built to attract sponsors, showcase his career, and grow his public profile in the motorsport world. Open-sourced as a portfolio example.

## 🔥 Features

- **🎬 Cinematic Hero** — Full-bleed race photography with bold typographic identity
- **📅 Career Timeline** — Race results, sponsorship history, and achievements
- **🤝 Sponsor Section** — Partnership CTA, media gallery, and sponsor contact form
- **📄 CV Download** — Media pack and CV for sponsor outreach

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Framework | React + Vite |
| Language | TypeScript |
| Styling | Tailwind CSS |

## 🚀 Getting Started

```bash
git clone https://github.com/tsvtnv/EmilHey.git
cd EmilHey
npm install
cp .env.example .env.local
npm run dev
```
